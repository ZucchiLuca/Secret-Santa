# 🎅 Secret Santa

A modern Secret Santa application built with Flutter that simplifies gift exchanges among friends, families, teams, and communities.

## Overview

Secret Santa is a cross-platform mobile application designed to automate participant management and gift assignments while preserving anonymity. The project focuses on delivering a simple and intuitive user experience through a clean Flutter-based architecture.

This application demonstrates the development of a complete mobile solution using Flutter, with attention to UI design, state management, and maintainable code structure.

---

## Features

* Create and manage Secret Santa groups
* Add and organize participants
* Automatic random participant assignment
* Anonymous gift exchange workflow
* Responsive and modern user interface
* Cross-platform support (Android, iOS, Web)

## Getting Started

### Prerequisites

* Flutter SDK
* Dart SDK
* Android Studio or VS Code
* Git

### Installation

```bash
git clone https://github.com/ZucchiLuca/Secret-Santa.git

cd Secret-Santa

flutter pub get

flutter run
```
